# Simple 3D Shooter Game

Game 3D shooter sederhana menggunakan Panda3D.

## Fitur
- 🎮 Player yang bisa bergerak di sekitar arena
- 🔫 Sistem shooting dengan bullet
- 🎯 Target yang bisa dihancurkan
- 📊 Score system
- ❤️ Health system

## Requirements
```bash
pip install panda3d
```

## Cara Main
```bash
python main.py
```

## Controls
- **WASD** - Bergerak (W: forward, A: left, S: backward, D: right)
- **Mouse** - Arahkan kamera
- **Left Click** - Shoot
- **ESC** - Keluar

## Game Mechanics
- Shoot target yang muncul di arena
- Setiap target yang dihancurkan = +10 poin
- Jangan sampai musuh mendekat terlalu jauh
- Lihat score di layar

Enjoy! 🎮
